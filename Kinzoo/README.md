## App Description
Kinzoo is a safe and secure messenger app built for kids and families to connect.

## Requirements
- iOS 14.5+
- Xcode 13.1+

## Running Kinzoo
- when launching the xcode project please use the `Kinzoo.xcodeproj` file.
